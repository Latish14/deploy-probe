What This Is

read me

This is a minimal FastAPI service built to sanity-check a self-hosting pipeline: point your platform at this repo, let it build the Dockerfile, and confirm the link it hands back actually serves a live, working app.

  • ✓ container builds & boots from docker-compose.yml
  • ✓ static assets served correctly
  • ✓ API round-trips respond (see signal panel above)
  • ✓ environment variables reach the container
  • ✓ service survives a page reload / restart
